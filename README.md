# 📋 Smart To-Do List

## 📖 About the Project

Smart To-Do List is a full-stack web application that helps users manage their daily tasks efficiently. It allows users to add, edit, delete, and organize tasks through a simple and user-friendly interface.

## ✨ Features

- User Login
- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Calendar View
- Progress Dashboard
- Habit Tracker
- User Profile
- Settings

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

## 🚀 How to Run

```bash
npm install
node server.js
```
